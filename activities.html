Prompt Poker

Quick Start

Goal: Build the best prompt hand to solve a problem. The game works like poker.

Setup: 2-4 players, 5 cards each.

Play: Optionally exchange up to 3 cards, build your prompt from your hand, then reveal and compare hands.

Prompt = Hand: Problem → Context → Main Point → Action → Output.

Winning: Pair = weak, Straight = complete prompt, Flush = same tool, Full House = strong, Straight Flush = very strong, Royal Flush = perfect workflow.

Key idea: Like poker, you do not need perfect cards. You need the best combination.
